 Para Execurar o compilador bosta rodar os seguintes comando:

 java -jar Coco.jar -frames .. uportugol.atg
 javac *.java
 java Compile <ARQUIVO_DE_ENTRADA> > <ARQUIVO_DE_SAIDA>

 Substituir <ARQUIVO_DE_ENTRADA> pelo nome do arquivo que contem o código que vai ser formatado.
 Substituir <ARQUIVO_DE_SAIDA> pelo arquivo HTML que será gerado contendo a formatação.
